Guess Secret Number



# Hint Box #
The secret number lies between 0 to 1000000

Trial Left : 15




Rules
  • You have got 15 trials to guess the correct secret number.
  • Enter any number which lies within the range of the numbers shown in the Hint Box.
  • For each number entered the number of trials is reduced by one (1).
  • For those entered numbers which lies outside the range , no trials are lost.
  • For each number entered which lies out of range a message is displayed notifying the end limits.
